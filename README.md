# Midi 2 Qwerty plus
(MIDI input to Qwerty output) для пианино, подключённого через кабель. Эмулирует нажатия клавиш на клавиатуре.
Поддержка 88 клавиш, силы нажатия (velocity) и проигрывание midi-файлов.
<br><br>
Проект вдохновлен [miditoqwerty](https://github.com/ArijanJ/miditoqwerty). Решил проверить смогу ли я реализовать то, что уже есть, но с дополнительным функционалом и более структурированным подходом.

## Описание Настроек
1. `Enable QWERTY output` - если включено, то будут эмулироваться нажатия клавиш клавиатуры при проигрывании нот.
2. `Enable velocity` - если включено, то будут эмулироваться нажатия клавиш, отвечающих за силу нажатия на клавиши на пианино.
3. `Enable sustain` - если включено, то будет нажиматься пробел, во время нажатия педали.
4. `MIDI input` - устройство для чтения нажатий на клавиши пианино (обычно это и есть пианино, подключенное через кабель).
5. `MIDI output` - устройство для проигрывания нажатых клавиш (опционально).
6. `Open MIDI file` - открывает файл из папки `songs` для его проигрывания.

## Использование
1. Для игры на пианино в реальном времени:
	1. Подключите пианино;
	2. Запустите программу и выберите в `MIDI input` пианино;
2. Для проигрывания MIDI-файлов:
	1. Переместите в папку `songs` желаемый `.mid` файл;
	2. Выберите в меню `Open MIDI file` данный файл и нажмите `F5`;
